`make clear` : Vide le cache

`make run` : Lance l'application en HTTP

`make route` : Liste les routes de l'API

`make create-migration` : Si de nouvelles entités sont ajoutées au dossier Entity
ou que des entités ont été modifiées, cette commande génère directement le SQL associé
pour effectuer ces modifications.

`make migrate` : Effectue les modifications des migrations enregistrées

`make create-db` : Initialise la base de données au début de l'application

`make install` : Installe les packages et initialise les fichiers nécessaires

`make dotenv` : Vérifie l'ordre des variables d'environnement
